📊 Training vs Testing

We never use all our data for training. We split it:

  • Training Set (80%): The data the model uses to learn patterns.
  • Testing Set (20%): The "final exam" to see how well the model performs on new data.

⚠️ Overfitting

Overfitting happens when a model learns the training data too well—including the noise and random errors. It becomes like a student who memorizes the textbook but can't solve a new problem.

Result: High accuracy on training data, but poor performance on real-world data.
