🎯 Model Evaluation

How do we know if a model is good? We use metrics like Accuracy. Accuracy is the percentage of correct predictions out of the total predictions made.

Accuracy = (Correct Predictions) / (Total Predictions)

⚖️ Bias vs Variance

This is the fundamental trade-off in Machine Learning. Finding the "Sweet Spot" is the goal of every data scientist.

High Bias

Underfitting: The model is too simple. It misses the patterns in the data entirely.

High Variance

Overfitting: The model is too complex. It fits the noise instead of the signal.
